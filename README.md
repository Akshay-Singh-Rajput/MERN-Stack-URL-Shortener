# Short_URL
MERN Project

- Paste Your Long Url In First Input feild
- And get a short URL In second Output feild
- Copy short URL and use it

## Tech Stack
- ### MongoDB      -> Database
- ### Express         -> API End Point/Routes
- ### React            -> Client side Render
- ### Node Js        -> Server side Render
- ### Chakra UI     -> CSS
- ### Heroku         -> Deployment
